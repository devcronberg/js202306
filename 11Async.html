    ASync

    Asynkron programmering er en metode til at håndtere operationer, der kan tage tid at udføre, såsom netværk og filsystem operationer, uden at blokere eksekveringen af resten af ​​programmet. På dette modul vil du lære om asynkrone metoder som callback-funktioner, promises og async/await i Javascript og hvordan man kan bruge dem til at håndtere asynkrone operationer.

    Generelt om asynkron kode

    Asynkron kode i JavaScript er typisk kodet med callback-funktioner:

    const setTimeoutPromise = (ms) =>
    new Promise((resolve) => setTimeout(resolve, ms));

console.log("start");
setTimeoutPromise(2000).then(function () {
    console.log("slut");
});

    Fra ES2017 er async/await dog en del af syntaksen, og det gør koden noget mere logisk og “synkron-agtig”. await (som skal placeres i en async funktion) afventer at promise-objektet er afviklet. Yderligere er fejlhåndtering simplificeret til simpel try/catch.

    const setTimeoutPromise = (ms) =>
    new Promise((resolve) => setTimeout(resolve, ms));

async function test() {
    console.log("start");
    let t = await setTimeoutPromise(2000);
    console.log("slut");
}

test();

    Flere promise-objekter kan eventuelt await’es med all

    const setTimeoutPromise = (ms) =>
    new Promise((resolve) => setTimeout(resolve, ms));

async function test() {
    console.log("start");
    const t1 = setTimeoutPromise(2000);
    const t2 = setTimeoutPromise(2000);
    await Promise.all([t1, t2]);
    console.log("slut");
}

test();

    AJAX

    AJAX (Asynkron JavaScript og XML) er den gamle betegnelse for den funktionalitet i browserne som gør det muligt at foretage HTTP kald.

    I ældre browsere skete dette ved hjælp af httpRequest-objektet, men i nyere browsere sker det ved hjælp af [fetch-API}(https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API). Med dette API kan du foretage alle former for HTTP kald relativt simpelt.

    Eksempel uden async/await

    index.html

    <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
    </head>
    <body>
        <button id="knap">Knap</button>
        <script src="app.js"></script>
    </body>
</html>

    app.js

    (function () {
    // hent alle kommuner i DK når der klikkes på knappen
    let knap = document.querySelector("#knap");
    knap.onclick = function () {
        fetch("https://dawa.aws.dk/kommuner/")
            .then((response) => {
                response.json().then((json) => {
                    console.log(json);
                });
            })
            .catch((error) => {
                console.log(error);
            });
    };
})();

    Eksempel med brug af async/await

    index.html

    <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
    </head>
    <body>
        <button id="knap">Knap</button>
        <script src="app.js"></script>
    </body>
</html>

    app.js

    (function () {
    // hent alle kommuner i DK når der klikkes på knappen
    let knap = document.querySelector("#knap");
    knap.onclick = async function () {
        try {
            let response = await fetch("https://dawa.aws.dk/kommuner/");
            let json = await response.json();
            console.log(json);
        } catch (error) {
            console.log(error);
        }
    };
})();
